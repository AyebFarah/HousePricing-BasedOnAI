# Prédiction des Prix Immobiliers en Tunisie

## Description du Projet

Ce projet vise à prédire les prix des biens immobiliers sur le marché tunisien en utilisant des techniques de Machine Learning. Les données sont collectées depuis tayara.tn, le plus grand site d'annonces immobilières en Tunisie, puis analysées et modélisées pour fournir des estimations précises.

## Objectifs

* Estimer la valeur d'un bien immobilier en fonction de ses caractéristiques

* Identifier les facteurs qui influencent le plus les prix

* Aider à la prise de décision pour les investissements immobiliers

* Fournir des prix compétitifs basés sur l'analyse du marché
